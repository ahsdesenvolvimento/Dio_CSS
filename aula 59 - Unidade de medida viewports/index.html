Unidade relativa: viewport units

Viewport é a área visivel de uma página web, podendo variar de dispositivo para dispositivo. As unidades de medida viewport, vão ser calculadas em relação ao tamanho da área em que o usuário esta vendo sua página.

Unidade de medida vw/vh/vmin/vmax

width: 50%;
width: 50vw;
width: 50vh;



Width: 50%;
Width: 50vw;
Width: 50vh;



Width: 10vmin;
Width: 10vmax;
